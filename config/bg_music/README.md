# Background music

Add **royalty-free** MP3 tracks here (you must own/license them).

| File | Use |
|------|-----|
| `ambient_cinematic.mp3` | Default subtle bed under narration (`pipeline.json` → `bg_music.track`) |

**Sources (examples):** YouTube Audio Library, Pixabay, Uppbeat — pick consistent "tense documentary / cinematic ambient" style to match competitor recaps.

The VPS mixer loops/trims the track to video length at low volume (~12%) so voice stays clear.
